## 2. Raw Data

### `data/raw/ana/` — 2,947 CSV files
One file per station, named by station code (e.g., `10100000.csv`). Downloaded from the ANA telemetric web service (`DadosHidrometeorologicos`). Each file contains timestamped sub-daily rainfall depths (mm) as read from tipping-bucket gauges.

### `data/reference/ibge/`
Official IBGE (Brazilian Institute of Geography and Statistics) geospatial layers.

## 3. Source Code (`src/huff_curves_br/`)

| Module | Role |
|---|---|
| `ana.py` | Downloads ANA telemetric data in 90-day chunks; handles retries and local caching. |
| `constants.py` | Project-wide defaults (IETD, depth/duration thresholds, date range, QC limits) and Huff reference polynomial coefficients (Quartiles 1–4, 7th-degree). |
| `series.py` | Timestep inference, series regularization, rainfall QC (negative values, intensity cap), and data-quality diagnostics. |
| `events.py` | Rainfall event extraction using IETD-based dry-gap splitting; filters on minimum depth, minimum records, maximum duration. |
| `huff.py` | Empirical Huff curves: cumulative curve per event, quartile assignment (dominant depth quarter), 7th-degree polynomial fit to median curve, KGE/R²/RMSE/MAE computation against original Huff reference. |
| `metrics.py` | Scalar goodness-of-fit metrics: KGE, NSE, IA, PBIAS, RMSE, MAE, R². |
| `pipeline.py` | End-to-end station workflow: download/cache → QC → event extraction → Huff fitting → CSV export. Supports parallel workers. |
| `geodata.py` | Loads IBGE layers; spatially joins station coordinates to municipality, state, region, and biome. |
| `regional.py` | Aggregates station-level median curves to state, municipality, and biome; fits regional 7th-degree polynomials; exports CSV, GeoPackage, and shapefile products. |
| `maps.py` | Optional choropleth/dot maps (dominant quartile, KGE, event count). |
| `reference.py` | Generic reference-layer downloader helper. |
| `stations.py` | Loads and normalizes the station catalog CSV. |

## 6. Primary Outputs

### Station-level CSVs (`outputs/`)

| File | Rows | Description |
|---|---|---|
| `station_huff_coefficients.csv` | 3,164 | One row per station. Fields: station metadata, QC status and reason, diagnostics (timestep, years, missing fraction, max daily depth), dominant quartile, per-quartile event counts, mean volume/duration/intensity, KGE/R²/RMSE/MAE, 8 polynomial coefficients (c1–c8), and `kge_mean` across quartiles. |
| `rainfall_events.csv` | 1,166,800 | One row per extracted event. Fields: station_id, event_id, start, end, duration_h, volume_mm, average_intensity_mm_h, maximum_intensity_mm_h, dominant_event_quartile, n_records. |
| `huff_curves_long.csv` | large | Long-form percentile curves (p10–p90 in 10% steps plus median) by station, quartile, and τ value on a 0.02 grid. |
