Brazil's first empirical Huff atlas

Huff (1967) curves classify storms by when peak intensity occurs. Q1 is front-loaded (peaks early); Q4 is back-loaded (peaks late). This atlas was derived from thousands of ANA sub-daily gauge records across all six Brazilian biomes.

Explore the map

  • Dot color shows the dominant quartile at each station; dot size scales with event count.
  • Filter by biome, state, or quartile in the left panel, or type a station code, city, or state name in the search box.
  • Click any dot on the map or any row in the station list to load its Huff curves in the right panel.

Build design storms

  • Under Design Storm in the right panel, set storm duration, total volume, and time step — then download the hyetograph as CSV.
